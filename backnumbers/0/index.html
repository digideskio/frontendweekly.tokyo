JavaScript Training Sucks — JavaScript Scene

Style Guide Best Practices — Brad Frost

Naming Things — 24 ways

In Other News…

Cohesive UX — 24 ways

Reactive MVC and the Virtual DOM — Futurice

An Effective Code Review Process — Jake McCrary
